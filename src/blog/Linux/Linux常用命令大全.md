# Linux常用命令大全

## 文件操作

### tree，生成文件目录树

#### 参数 

| 参数 | 说明                                     | 用法                                |
| ---- | ---------------------------------------- | ----------------------------------- |
| -I   | 忽略                                     | `tree -I "node_modules|dist"`       |
| -L   | 指定层级                                 | 如只想要生成两级的目录，`tree -L 2` |
| -C   | 在文件和目录上添加色彩，以便区分各种类型 | `tree -C`                           |
