# VSCode 快捷键

| 快捷键             | 说明                                     |
|--------------------|------------------------------------------|
| control + ~        | 打开终端                                 |
| option + shift + f | **格式化**                               |
| command + D        | 选中下一个相同的内容                     |
| command + F        | 搜索                                     |
| control+R          | 打开最近的项目列表                       |
| command + G        | 逐行搜索                                 |
| command+B          | 开启/关闭菜单栏                          |
| command + p        | 以列表形式展示当前目录下的文件，可以搜索 |
| command+W          | 关闭当前窗口                             |
| command + K + W    | 关闭当前打开的所有标签                   |
| command+O          | 在当前窗口打开文件/文件夹                |
