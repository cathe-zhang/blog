# 前端开发环境搭建

## 基础环境

- Node.js
- Git

## 开发工具

- VSCode
  - 插件/主题
    - chester atom | 编辑器主题
    - Live Server | 开发服务器
    - Filesize | 显示当前文件体积
    - Vetur | Vue语法提示
    - Beautify | 代码格式化
- 微信开发者工具 | 微信公众号/小程序开发

## 测试工具

- Whistle | 抓包
- Posterman | 接口调试

## 提效工具

- nrm | 切换 npm 源
- nvm | 切换 Node.js 版本
